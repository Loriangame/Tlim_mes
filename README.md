# Tlim Messenger

Полноценный мессенджер с реальным временем, созданный для деплоя на Render.com.

## Возможности
- 🔐 Регистрация и вход
- 💬 Личные чаты
- 👥 Группы
- 📢 Каналы
- 📎 Отправка файлов
- 🎥 Видео и кружки
- 🤖 DeepSeek Bot

## Деплой на Render

1. Загрузите проект на GitHub
2. На Render.com создайте новый Web Service
3. Подключите репозиторий
4. Настройки:
   - Build Command: `npm install`
   - Start Command: `node server.js`
   - Node Version: 16+

## Локальный запуск

```bash
cd backend
